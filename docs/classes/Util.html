  • tagName
)
Element | Array

Defined in hncollapse.user.js:402

Return the parent element based on tagName. If a list is passed, return a list of parent elements treating each source element as if it had been passed individually.

Parameters:

  • element Element | NodeList

    The child element or list of child elements

  • tagName String

    HTML tag name of the parent element to find. May can also include a className to further refine search. ex. "tr", "tr.comment"

Returns:

Element | Array:

parent element or list of parent elements. If none found, returns undefined or list of undefined
